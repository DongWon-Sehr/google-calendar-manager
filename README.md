## Prepare
1. Google Sheet Setting
>- User input data in "Data" sheet
>- The "Data" sheet will be synced to "Birthday" sheet and "Welcome day" sheet
>- The main code "code.js" will read data from "Birthday" sheet and "Welcomde day" sheet
><img width="1000" alt="image" src="https://github.com/DongWon-Sehr/google/blob/master/sample_image/01_Google%20Sheet%201.png?raw=true">

><img width="1000" alt="image" src="https://github.com/DongWon-Sehr/google/blob/master/sample_image/03_Google%20Sheet%203.png?raw=true">

2. Google Calendar ID
>- Can call specific Google Calendar with Calendar ID
><img width="1000" alt="image" src="https://github.com/DongWon-Sehr/google/blob/master/sample_image/04_Google%20Calendar%20ID.png?raw=true">

3. Calendar API (convert lunar to solar calendar)
>- Calendar API for people who use lunar calendar birthday (Google Apps Script or Google Calendar don't support lunar calendar)
